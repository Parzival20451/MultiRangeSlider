## [1.0.0] - 2026-05-19

### This is the first release of MultiRangeSlider

### Added

- MultiRangeAttribute that accepts min-max or max-min ranges
- Specific editor drawers for MinMax and MaxMin sliders
- Works with multiple selected objects
- Custom structs RangeValue, RangeValueInt with min and max properties for better code readability
- Integration with Vector2 and Vector2Int too
